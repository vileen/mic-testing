# 🎤 Mic Testing

Real-time microphone loopback testing tool with Discord/Teams simulation.

**Live demo:** https://vileen.github.io/mic-testing

## Features

- 🔴 **Continuous loopback** — talk and hear yourself instantly
- ⏱️ **Adjustable delay** (0-5s) — simulates VoIP latency
- 🎚️ **Voice compression** — like real Discord/Teams processing
- 🎛️ **Noise gate** — cuts silence automatically
- 📊 **VU meter** — visual input level
- 🎧 **Presets** — Discord, MS Teams, or raw audio

## Usage

1. Open https://vileen.github.io/mic-testing
2. Click "Start Mic Test"
3. Allow microphone access
4. **Use headphones** to prevent feedback!
5. Adjust delay/gain in real-time

## Local Development

```bash
git clone https://github.com/vileen/mic-testing.git
cd mic-testing

```
